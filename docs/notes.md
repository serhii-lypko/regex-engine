Regular expressions describe regular languages in formal language theory. 
They have the same expressive power as regular grammars. But the language of 
regular expressions itself, is context-free language.


# Building blocks:

Alphabet - your set of valid characters
Language - set of valid strings over that alphabet
Finite Automaton (FA) - state machine that accepts/rejects strings
Regular Expression - pattern notation describing languages
Equivalence - FAs and regexes describe the same class of languages
